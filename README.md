# 🌍 Anthropology Telegram Bot

بوت تيلجرام موسع متخصص في علم الإنسان (الأنثروبولوجيا العالمية والجزائرية) ومبني بلغة بايثون.

## ✨ المميزات:
- **أزرار ريبلي تفاعلية (ReplyKeyboardMarkup)** مدمجة داخل حقل الكتابة لتصفح شجري نظيف.
- **محتوى موسع وعالمي** يشمل الأنثروبولوجيا الثقافية، الحيوية، اللغوية، وعلم الآثار.
- **قسم خاص بالجزائر (الدزاير) 🇩🇿** يستعرض عمقها التاريخي ومواقعها الأثرية العالمية مثل عين الحنش والطاسيلي.
- **نظام جدولة تلقائي (JobQueue)** ينشر معلومات عشوائية 3 مرات يومياً للمشتركين.
- جاهز تماماً للنشر الفوري على منصة **Railway**.
